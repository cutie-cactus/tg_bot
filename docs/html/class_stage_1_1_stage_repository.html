Detailed Description

Репозиторий для управления состояниями пользователя в Telegram-боте

Обеспечивает сохранение и восстановление:

  • Текущего окна (WindowType)
  • Стадии диалога (StageType)
  • Идентификаторов активных событий и напоминаний
