Detailed Description

Класс для настройки и управления логированием приложения.

Обеспечивает запись логов как в файл (указанный в конфигурации), так и вывод в консоль. Поддерживает все стандартные уровни логирования.

Logger.Logger.__init__ ( self)

Инициализирует логгер с настройками из конфигурационного файла.

Создает:

  • Файловый обработчик (FileHandler) для записи в файл логов
  • Консольный обработчик (StreamHandler) для вывода в stdout Оба обработчика используют формат: '%(asctime)s - %(levelname)s - %(message)s'
